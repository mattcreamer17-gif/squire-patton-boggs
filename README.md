# Squire Patton Boggs, private briefing page

A single-purpose landing page built for one conversation at eMerge Americas Miami 2026. The page renders in the firm's own visual vocabulary: ivymode serif for display, arboria sans for body, charcoal-navy (#182935) with teal (#05D8C8) as the interactive accent. Matt Creamer walks the booth with this page on a phone and hands it to a Knowledge Management director, a CIO, or a practice-group AI lead. The copy is a read on where an AI orchestration layer fits above iManage.

Intended URL path: /for/squire-patton-boggs

## Files
- `index.html`, single static page, no JS
- `styles.css`, brand-token CSS
- `logo.svg` and `logo.png`, firm mark (black SVG primary, color PNG fallback)

## CTA
All CTAs resolve to `https://calendly.com/thecro-eduba/30min` (Matt Creamer, 30 minutes).
